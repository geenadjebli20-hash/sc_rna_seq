# sc_rna_seq
Structure du pipeline de filtration construit par moi et utilisé pour analyser des données de single cell Rna seq chez l'embryon d'oursin. (Stade 8 , 16 cell et mesanchyme blastula)  
Pour l'instant ce sont les piplines sur les cellules fixées. 
Tout les étapes sont décrites dans les fichiers html.
